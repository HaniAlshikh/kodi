Installing the H-S repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager
  • Click on "Add source"
  • The path for the source is https://hanialshikh.github.io/kodi/packages
  • Give it a name for example "H-S"
  • Head to "My Addons"
  • Select install from zip file.
  • When it asks for the location, select "H-S" (or whatever you called it)
  • Click on repository.hs.zip to install.
